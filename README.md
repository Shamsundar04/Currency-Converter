💱 Currency Converter Web App

A simple and responsive Currency Converter web application built using HTML, CSS, and JavaScript. This app allows users to convert one currency into another using real-time exchange rates.

🚀 Features
🌍 Convert between multiple currencies
🔄 Real-time exchange rate updates (via API)
💻 Clean and responsive UI
⚡ Fast and lightweight (no frameworks)
🎯 Easy to use interface
🛠️ Tech Stack
HTML5 – Structure of the app
CSS3 – Styling and layout
JavaScript (Vanilla JS) – Functionality and API handling

📂 Project Structure
currency-converter/
│
├── index.html        # Main HTML file
├── style.css         # Styling file
├── script.js         # JavaScript logic
└── README.md         # Project documentation
⚙️ How It Works
User selects base currency
User selects target currency
Enters the amount
App fetches exchange rate from API
Displays converted value instantly
🔌 API Used

You can use any exchange rate API, such as:

ExchangeRate API
